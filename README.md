# SimplePixel

`SimplePixel` is a browser-based 3D pixel-art object editor for quickly creating game-ready voxel assets.

## Features

- Basic primitive creation: cube, sphere, cylinder, cone, stair
- Editing tools: pen, erase, fill, gradient
- Sculpting tools: melt, weld, smooth
- Selection workflow with stamp library (save and reuse pieces)
- Toggleable lighting preview with adjustable direction
- Downloads as `.json` (native voxel format) and `.obj` (mesh export)

## Run locally

1. Open `index.html` in any modern browser.
2. Or serve the folder with any static server and open the shown URL.

## Controls

- Left click: apply active tool
- Right click: quick erase while on pen
- Shift + drag (or middle click + drag): orbit camera
- Hotkeys: `Q` pen, `W` erase, `E` select, `R` stamp
